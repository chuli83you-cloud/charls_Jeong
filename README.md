# 블로그 글 → 스레드 5개 생성기

블로그에 작성한 글을 붙여 넣으면, 핵심 키워드를 추출해 **서로 다른 주제의 스레드 5개**를 자동으로 만들어주는 웹 앱입니다.

## 실행 방법

```bash
python app.py
```

실행 후 브라우저에서 `http://localhost:8000`으로 접속하세요.

## 기능

- 블로그 원문 입력
- 자동 주제 추출 (5개)
- 주제별 스레드 초안 생성
- 한국어 블로그 글 기준으로 빠르게 초안 생성

## 참고

- 생성 결과는 초안이므로, 발행 전 문체/사실관계는 꼭 검토해 주세요.
